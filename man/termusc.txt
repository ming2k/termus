@title TERMUSC 1 05/11/2006 termus

@h1 NAME

termusc - control termus


@h1 SYNOPSIS

termusc [*OPTION*]... [`FILE`|`DIR`|`PLAYLIST`]...@br
termusc *-C* `COMMAND`...@br
termusc


@h1 DESCRIPTION

Add `FILE/DIR/PLAYLIST` to playlist, library (*-l*) or play queue (*-q*).

If no arguments are given termusc reads raw commands from stdin (one
command per line).  Raw commands are termus' command mode commands.  These same
commands are used in configuration files and key bindings.  *termus*(1) contains
full list of commands.  For consistency also searching is supported:
*-C /text*.

If *-C* is given, all command line arguments are treated as raw commands.

@h1 OPTIONS

--server SOCKET
	Connect using socket *SOCKET* instead of `$XDG_RUNTIME_DIR/termus-socket`.

--passwd PASSWD
	password to use for TCP/IP connection

--help
	Display usage information and exit.

--version
	Display version information and exit.

-p, --play
	Start playing.

-u, --pause
	Toggle pause.

-U, --pause-playback
	Pause if currently playing.

-s, --stop
	Stop playing.

-n, --next
	Skip forward in playlist.

-r, --prev
	Skip backward in playlist.

-R, --repeat
	Toggle repeat.

-S, --shuffle
	Toggle shuffle.

-v, --volume VOL
	Change volume. See *vol* command in *termus*(1).

-k, --seek SEEK
	Seek. See *seek* command in *termus*(1).

-f, --file FILE
	Play from file.

-Q
	Get player status information.  Same as *-C status*.  Note that
	*status* is a special command only available to termusc.

-l, --library
	Modify library instead of playlist.

-P, --playlist
	Modify playlist (default).

-q, --queue
	Modify play queue instead of playlist.

-c, --clear
	Clear playlist, library (*-l*) or play queue (*-q*).

-C, --raw
	Treat arguments (instead of stdin) as raw commands.

@h1 REMOTE COMMANDS

Special commands only available to termusc.

status
	Print information about currently playing track.

format_print
	Print arguments as `Format Strings`. Each argument starts a new line.

@h1 EXAMPLES

Add playlists/files/directories/URLs to library:

	@pre
	$ termusc -l music.m3u \
	http://live.urn1350.net:8080/urn_high.ogg
	@endpre

Load (clear and add) playlist to playlist view:

	@pre
	$ termusc -c music.m3u
	@endpre

Three different ways to toggle repeat:

	@pre
	$ termusc -R
	$ termusc -C "toggle repeat"
	$ termusc
	toggle repeat
	^D
	@endpre

Query settings or key bindings:

	@pre
	$ termusc -C "set repeat?"
	setting: 'repeat=false'
	$ termusc -C "showbind common a"
	bind common a win-add-l
	@endpre

Dump the playlist to stdout:

	@pre
	$ termusc -C "save -p -"
	[...]
	@endpre

Search works too:

	@pre
	$ termusc -C /beatles
	@endpre
